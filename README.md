# Sudoku – README

## 📘 O que é Sudoku?

Sudoku é um jogo de lógica numérica no qual o objetivo é preencher uma grade 9×9 com números de 1 a 9.  
Cada número deve aparecer **apenas uma vez** em:
- Cada linha
- Cada coluna
- Cada bloco 3×3

Não é necessário fazer cálculos — apenas lógica e dedução.

---

## 🧩 Regras do Jogo
1. A grade começa parcialmente preenchida.
2. Você deve completar os espaços vazios.
3. Regras obrigatórias:
   - Não repetir números na **linha**.
   - Não repetir números na **coluna**.
   - Não repetir números no **bloco 3×3**.
4. O jogo termina quando todos os espaços estiverem corretos.

---

## 🎯 Objetivo
Completar toda a grade 9×9 seguindo as regras do jogo, usando apenas dedução lógica.

---

## 🧠 Estratégias Comuns
### ✔ Checagem simples (Single)
Quando um número só pode estar em uma única célula dentro de uma linha, coluna ou bloco.

### ✔ Candidatos possíveis
Anote mentalmente (ou no próprio jogo) quais números podem estar em cada célula.

### ✔ Procure por padrões
Algumas combinações de candidatos eliminam possibilidades em outras células.

---

## 🕹 Como Jogar
1. Observe as pistas já preenchidas.
2. Verifique as restrições de cada célula.
3. Preencha com segurança — evite “chutar”.
4. Continue eliminando impossibilidades até resolver todo o puzzle.
